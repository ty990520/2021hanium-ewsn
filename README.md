```

```
3️⃣ Heroku 클라우드 서버를 대여하여 서비스를 배포하였습니다.

### 🛠️ Specification

- Java
- Spring MVC
- JavaScript
- ajax
- REST API
- Git, Git Flow

### 🧐 담당 역할

- 박태영
  - 개발 팀장
  - DA 단계적 보안성 평가, 회원관리, 관리자 메뉴
  - Heroku 어플리케이션 배포
- 김혜진
  - 사이버 보안 통제항목 관리, 신규 취약점 관리, 취약점조치권고, 이행계획
- 박민지
  - DA정보 관리, 통제항목 관리, 대표코드
