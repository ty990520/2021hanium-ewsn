자주 묻는 질문

대부분의 문제에 대한 해결책을 아래 항목에서 찾을 수 있습니다. 그 밖의 항목에 대해서는 피드백을 주세요.



1. 자산 보안성 평가의 절차는 어떻게 되나요?


자산 보안성 평가의 절차는 평가 방식에 따라 'EP -> BOP -> INDIRECT'로 이루어지며 한 방향 평가 방식 입니다.
따라서 평가 도중에는 중간 저장이 불가능하며 평가 도중 뒤로 넘어갈 수 없습니다.



2. 자산 단계적 보안성 평가는 중간 저장이 불가한가요?


단계적 보안성 평가는 EP, BOP, Indirect를 한번에 분류하기 때문에 도중에 그만둘 수 없으며,
사이트를 나갈경우 이전까지 평가한 내용은 저장되지 않습니다.



3. 일반 사용자는 어디까지 권한이 허용되나요?


일반사용자는 자산정보관리, 통제항목 관리, 신규 취약점 관리, 조치권고 이행계획에서 수정만 가능하며,
나머지 기능(단계적 보안성 평가, 통제항목 적용관리, 조치권고)에서 수정, 삭제 둘 다 불가합니다.



4. 관리자가 내가 등록한 계획을 수정할 수 있나요?


관리자는 일반사용자가 등록한 계획을 수정, 삭제할 수 없습니다.
이는 조치권고 이행계획 등록 내역을 지속적으로 관리하기 위함입니다.



5. 추가 기능을 요청하고 싶어요.


추가 기능을 요청하고 싶다면 우리에게 연락을 해주십시오. 내부 회의에 걸쳐 적극적으로 반영하겠습니다.
이메일 주소는 EWSN123@gmail.com 입니다.



6. 회원정보를 잃었어요.


우리 프로그램은 일반 사용자가 회원정보를 수정하는 것 이외에 회원정보 찾는 것을 불가능하게 설계 되어있습니다.
따라서 회원정보를 잃었다면 관리자에게 직접 연락을 주십시오. 관리자 이메일 주소는 EWSN_Manage@gmail.com 입니다.
